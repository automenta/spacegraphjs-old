Template

This is a template for making a new demo. Copy all files in this folder to a new demo folder and start making your own ammo.js demo!

  1. Copy the _template folder, name it something descriptive eg. MyDemo.
  2. Rename TemplateDemo.js to something like MyDemo.js. Remember to change the include src in index.html!
  3. Customize MyDemo.js as you want, produce a cool demo.
